## Tip:

本项目前后端均部署在 Vercel, 由于 Vercel 在国内遭到了 DNS 污染。本项目采取了自定义域名的方式，使国内可以顺利访问到前后端。

前端地址：https://music.imchin.top

后端API地址：https://music-api.imchin.top
